# 🚀 Projetos

## 💰 Finanças

### Aplicativo de Controle Financeiro

Aplicação de gestão financeira doméstica que começou como um **projeto em Python + Streamlit** e evoluiu para uma **Progressive Web App (PWA)** completa construída com React, Tailwind e Supabase.

O aplicativo gerencia contas mensais, acompanha pagamentos, gera relatórios formais em PDF e oferece análise financeira por meio de um **dashboard analítico estilo BI**.

Foi projetado para rodar totalmente **client-side**, utilizando apenas **Supabase (plano gratuito) + GitHub Pages**, mantendo a infraestrutura simples sem abrir mão de relatórios, análise de dados e funcionalidades de PWA.

## 🎲 Hobbies

### Web Novel Scraper

Pipeline automatizado que extrai capítulos de web novels e os compila em **eBooks .docx estruturados**.

Construído com **Streamlit + Selenium (undetected_chromedriver)** para contornar proteções como Cloudflare. O scraper navega automaticamente capítulo a capítulo, aplica regras personalizadas de normalização de texto, remove censuras artificiais e gera um documento final limpo e formatado.

### Gerador Procedural de Mundo RPG

Experimento em larga escala que combina **lógica, dados estruturados e LLMs** para gerar um cenário completo de campanha de RPG.

O projeto inclui:

- **140+ NPCs** organizados em facções e divisões sociais  
- um **grafo de relacionamentos** conectando todos os personagens  
- um **mapa interativo da cidade** mostrando onde cada NPC reside  
- overlays e filtros de facções no mapa  
- fichas de personagens exibidas dinamicamente na interface  
- um **livro de campanha de 150 páginas** com lore, imagens e descrições detalhadas  
- um **site de exploração interativa do cenário**

## 🧰 Utilitários

### Editor de Tradução com IA

Ferramenta local de IA projetada para **revisar e aprimorar traduções automáticas de web novels**, preservando tom, terminologia e estrutura narrativa.

Construída como um pipeline modular utilizando LLMs locais (Hermes / OpenHermes) rodando em GPU. O sistema processa capítulos em blocos estruturados e aplica controle rígido de prompts para garantir melhorias de clareza sem alterar o significado original.

Principais recursos:

- segmentação em blocos para revisão contextual
- estrutura de prompt rigorosa para evitar alucinações
- limpeza automática de artefatos gerados por LLM
- fallback automático quando uma revisão falha
- logs detalhados de desempenho

Repo  
[[link](https://github.com/RomanBrocki/editor_ia)]

### Ferramenta de Manipulação de PDF

Aplicação para manipulação de PDFs disponível tanto como **ferramenta web em Streamlit** quanto como **aplicação desktop local em Python**.
